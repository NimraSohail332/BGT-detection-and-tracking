Requirements
To run this code, you'll need to install several Python libraries and have a compatible environment. Here's a list of required libraries and tools:
Python Version
•	Python 3.8 or newer
•	Required Python Libraries
•	You can install all required libraries using pip.
these are the necessary libraries :
•	opencv-python==4.8.0.76
•	numpy==1.25.1
•	ultralytics==8.0.89
YOLOv8 Model
You need to have the YOLOv8 model weights file. Update the path in the code (model = YOLO('path/to/your/model.pt')) to the location of your YOLOv8 model file. If you don't have a pre-trained model, you can obtain it from the Ultralytics YOLOv8 repository.
OpenCV and Background Subtraction
The cv2 library is used for various computer vision tasks, including background subtraction and tracking. Ensure you have OpenCV installed.
Usage
Connect your webcam or ensure a video source is available.
Run the main script:
python main.py
The script will open a window showing the live video feed with object detection and tracking. Press q to exit the application.
